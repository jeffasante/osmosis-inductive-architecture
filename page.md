Thesis

Osmosis: Amortized Program Inference with GPU-Scale Search for Efficient Interactive Reasoning

Efficient intelligence in interactive environments emerges from amortized continuous inference over a discrete causal program space, where latent geometry is aligned with program behavior and massively parallel execution closes the loop.

We propose that sample-efficient reasoning is not achieved by larger sequence models, but by:
- learning a posterior over causal programs
- aligning its geometry with behavioral equivalence
- and using GPU-parallel execution to verify and refine hypotheses




Osmosis reframes reasoning as probabilistic program inference under compute constraints, where learning shapes the hypothesis space and search performs the final epistemic step.
